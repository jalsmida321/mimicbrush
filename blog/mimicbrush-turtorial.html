1. MimicBrush Quick Use Tutorial

Local Deployment (ComfyUI)

Requirements:

  • Windows OS
  • GPU with 8GB+ VRAM
  • CUDA 12.1

Installation:


1. Install via ComfyUI Manager
2. Restart to download models (~10GB)
3. Right-click image → "Open in Mask Editor"

Cloud Deployment (Xinghai Platform)


Operation Flow:
1. Create GPU instance
2. Select MimicBrush image
3. Upload source image
4. Draw editable area
5. Upload reference image
6. Check "Keep original shape"
7. Click Run Generate

Key Parameters:

  • if_keep_shape: Preserve structure
  • Weight: 0.5-1.2 (style strength)
